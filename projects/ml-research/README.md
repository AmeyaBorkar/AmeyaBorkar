# ml-research

Machine-learning and research projects — vision, audio, NLP/emotion, document binarization, vision-language models, and LLM memory. Mostly PyTorch.

Each linked doc below was written by **reading the actual source**, not the README — with computed line counts, verified parameter counts where possible, and an explicit "README vs. code" section. These projects had the largest gap between claims and implementation, so the "what's actually wired up" notes matter.

| Project | What it actually is | Stack / size | Notable code-level finding |
|---------|---------------------|--------------|----------------------------|
| [PROMETHEUS](PROMETHEUS.md) | A **document-image binarization ablation** on DIBCO — testing whether "fancy ideas" actually help | PyTorch · Neural ODE · DINOv2 · 29,635 LOC | All four fancy ideas (torchdiffeq ODE side-branch, FFT branch, frozen DINOv2 ViT-B/14, ASPP) are genuinely implemented. **Headline confirmed in code + stored results:** the physics-informed ODE branch *hurts* (−0.079 dB); the real win was dense 256×256 patch extraction (+2.72 dB). DRD and pseudo-F-measure are configured but never actually computed. |
| [AdaptaNet](AdaptaNet.md) | A **deformable-CNN + MoE-transformer** framework with many custom losses | PyTorch · 14,276 LOC | "20+ losses" is really **12** implemented criteria. The rich multi-loss / two-view / curriculum pipeline is built and unit-tested but **never wired into `train.py`** (actual training is plain cross-entropy or one ArcFace head). ~30% of the transformer code is dead. Hand-rolled `DeformableConv2d` via `grid_sample`. |
| [SGEE](SGEE.md) | **Semantically-grounded emotion embeddings** — anchors RoBERTa in VAD psycholinguistic space | PyTorch · RoBERTa · NRC-VAD · 24,390 LOC | RoBERTa `[CLS]` + 17-d NRC-VAD + 16-d surface features → 512-d embedding → 28-class + VAD heads. Stage 1 supervised, Stage 2 InfoNCE prototype-contrastive — plus an **undocumented Stage 3** (persuasion, DistilBERT distillation). VAD targets are lexicon-derived, not gold. Macro-F1 ~0.47–0.50. |
| [EdgeCAAI-Net](EdgeCAAI-Net.md) | A **1.42M-param music-genre classifier** that exposes artist leakage in benchmarks | PyTorch · torchaudio · 4,566 LOC | Param count **verified by instantiation: 1,418,217**. librosa is decode-only; mel spectrogram is torchaudio. The leakage finding is real and measured: track-disjoint splits leak **559 of 765 test artists**, dropping macro-F1 0.560→0.455. The gradient-reversal artist-invariance is the *external* classifier; the model's own `artist_head` is dead code. |
| [Engram](Engram.md) | An **LLM memory that consolidates** (raw events → summaries → abstractions) — published as `engrampy` | Python · PyPI `engrampy` 0.3.0 · 18,536 lib LOC | Decay is a closed-form recurrence `w·α^Δt + β·r + γ·c − δ·x` (30-day half-life). Three conceptual layers (the `Level` enum actually has six). Clustering via HDBSCAN/agglomerative; consolidation is optional/off-by-default. SQLite + **in-process numpy vector index** (no FAISS/pgvector). Providers: OpenAI / Anthropic / local sentence-transformers. Postgres + RLS is roadmap-only. |
| [LocateVision](LocateVision.md) | A **ResNet-50 + Swin gated-attention** campus image classifier with a web UI | PyTorch + HTML · 4,566 LOC (~551 Py) | The fusion model (2048-d ResNet + 1024-d Swin → sigmoid-gated → 512-d head) is real and matches the README. But the **99.76% accuracy is unverifiable** — it appears only as a string; no weights, dataset, logs, or notebook compute it. The frontend calls ~8 endpoints; the server implements one (`/predict`). |
| [Tesseract](Tesseract.md) | A **cargo X-ray vision-language model** for customs officers | FastAPI · Qwen2.5-VL-7B · 1,135 Py LOC | The VLM is **inference-only and remote** — `Qwen/Qwen2.5-VL-7B-Instruct` via `huggingface_hub.InferenceClient` (no local weights, no torch, no fine-tuning). "X-ray specialization" is **prompt-only**; the score/level/objects are reconstructed from free text by `risk_scorer.py` regex. 3 endpoints + a vanilla-JS dashboard. README claims MIT but there's no LICENSE. |
