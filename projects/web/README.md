# web

Front-end, web tooling, and zero-/low-dependency browser projects — a geospatial engine, a markdown viewer, a font specimen tool, and a production marketing site.

Each linked doc below was written by **reading the actual source**, not the README — with computed line counts and an explicit "README vs. code" section.

| Project | What it actually is | Stack / size | Notable code-level finding |
|---------|---------------------|--------------|----------------------------|
| [proximap](proximap.md) | A **headless OpenStreetMap geospatial engine** — *not* a map plugin, despite its `MapsPlugin` directory name | TypeScript · ~7,057 LOC · npm `@proximap/core` \| `cli` \| `mcp` 1.0.0 | Dependency-free core over Nominatim/Overpass/Valhalla/OSRM (key-free, haversine fallback). Features: nearby ranking, NL category resolver, walkability score, gap detection, isochrones, a **Generalized-TSP errand planner using exact Held-Karp DP**, and a stdio **MCP server**. README is highly accurate; `apps/web` and `python/` are honestly-labeled placeholders. |
| [markdown-viewer](markdown-viewer.md) | One Python package, a **hand-written markdown engine** (no external lib), three surfaces | Python · 3,632 LOC · `markdown-viewer` 0.1.0 | "No external markdown library" is **verified** (`dependencies = []`); the engine is a line-based single-pass block scanner with a stash/restore inline pass (tables, task lists, math/mermaid passthrough). Caveat: the desktop app has its *own* second renderer, so "same engine drives all three" is literally true only for API + web + export. 44 unit tests. |
| [FontVisualizer](FontVisualizer.md) | A **font specimen / comparison tool** in a single HTML file | Vanilla JS · one 1,253-line `index.html` | Live-previews **132 Google Fonts** with category filter, search, size slider, and a luminance-aware background switcher. Fonts load via Google Fonts CSS2 + `IntersectionObserver` lazy injection (no `FontFace`, no canvas text). Adds a drag-drop collection tray and a full-screen Studio Board (HSV wheel, lighting presets). No custom-font upload despite the name; no README. |
| [Elegant-Landscape](Elegant-Landscape.md) | The **live marketing site** for a Pune landscape studio ([elegantlandscape.in](https://elegantlandscape.in)) | Vanilla HTML/CSS/JS · ~4,740 LOC · zero build step | The **192-frame** WebP scroll-scrubbed `<canvas>` hero is confirmed (`TOTAL_FRAMES=192` + exactly 192 files), with key-frame-first preload then lazy fill, rAF-throttled. Zero build step confirmed (no `package.json`). Discrepancies: founding year (prose "2022" vs JSON-LD `2023-06-23`), a Google Fonts `@import` the "no dependencies" claim omits, and no `prefers-reduced-motion` handling. |
